
# 1. Layoffs Dataset — Data Cleaning with SQL
## Project Overview
- This project focuses on cleaning and preparing a global layoffs dataset using SQL.
- The dataset contains company layoff information across industries, locations, and time periods.
- The goal of this project is to transform raw data into a clean, structured, and analysis-ready dataset.
- Remove Duplicates
- Standardize Data
- Null Values or Blank Values
